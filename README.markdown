

SEGmeter 2.0:

Arduino-equivalent pin assignments:

0: Serial RX to AVR
1: Serial TX from AVR.
2: Spare, not used.
3: Status indicator LED output.
4: Pushbutton control input.
5: OneWire sensor bus.
6: Relay output 1.
7: Relay output 2.
8: Relay output 3.
9: SPI chip select for ADC.
10: Spare, not used. SPI chip select for ADC on expansion board.
11: SPI MOSI
12: SPI MISO
13: SPI clock

A0: Spare, not used.
A1: Spare, not used.
A2: Spare, not used.
A3: Spare, not used.
A4: Spare, not used.
A5: Spare, not used.
